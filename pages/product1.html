J.I.M.M

The Janssen Interactive Multimedia Module is a dedicated app for a 90-inch touchscreen display for the Engineering Building at the University of Idaho. JIMM scrapes the university's website for important data (faculty/staff, instructor schedules, campus maps, student clubs/organizations, etc.) and presents it to the user as a touch-optimized user interface that can be split into 3 separately functioning modules to allow for multiple users on the same screen.

Technologies Used: C#, Python, Unity
